# 🎨 Random Background Color Generator

A simple and interactive web application built using **HTML, CSS, and JavaScript** that dynamically changes the background color at regular intervals.

---

## 📌 Features

* 🎲 Random HEX color generation
* ▶️ Start button to begin color changing
* ⏹ Stop button to stop color changing
* ⏱ Automatic color change every 1 second
* 🎨 Smooth background transition effect
* 💡 Clean and modern UI design

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

---

## 📷 How It Works

1. User clicks the **Start** button
2. App generates a random HEX color
3. Background color changes every second using `setInterval`
4. User clicks **Stop** to stop the color changes using `clearInterval`
